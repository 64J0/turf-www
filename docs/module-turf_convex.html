turf.convex

Takes a set of points and returns a convex hull polygon.

Internally this implements a Monotone chain algorithm.

Parameters

Name Type Description
points FeatureCollection

a collection of Features with Point geometries

Returns

Feature

output a feature with Polygon geometry

Example

var fs = require('fs')
var pts = JSON.parse(fs.readFileSync('/path/to/pts.geojson'))
var hull = turf.convex(pts)
console.log(hull)
