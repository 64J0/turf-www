turf.extent

Calculates the extent of all features and returns a bounding box.

Parameters

Name Type Description
input GeoJSON-Object

any valid GeoJSON Object

Returns

Array.<number>

extent - the bounding box of the GeoJSON given as an array in WSEN order.

Example

var extent = require('turf-extent')
var fs = require('fs')
var fc = JSON.parse(fs.readFileSync('/path/to/myFeatureCollection.geojson'))
var bbox = extent(fc)
console.log(bbox) // [minX, minY, maxX, maxY]
