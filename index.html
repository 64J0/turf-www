Examples Source
Turf is a modular GIS engine written in JavaScript. It performs geospatial processing tasks with GeoJSON data and can be run on a server or in a browser. Here is an example in Node.js: {% highlight js %} // find all the snakes within 1 mile of a wombat var turf = require('turf'); var fs = require('fs'); var wombats = JSON.parse(fs.readFileSync('./wombats.geojson')); var snakes = JSON.parse(fs.readFileSync('./snakes.geojson')); var oneMileRing = turf.buffer(wombats, 1, 'miles'); var snakesNearWombats = turf.within(snakes, oneMileRing); console.log(snakesNearWombats); {% endhighlight %}

browsers

Use browserify to manage and include Turf as a node-style module, or the Mapbox CDN distribution of Turf for a traditional script-tag include.
