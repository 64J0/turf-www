Getting Started


Turf can be used as a framework, or as a collection of tiny modules.

framework

{% highlight sh %} npm install turf {% endhighlight %} ...then call functions like this {% highlight js %} var turf = require('turf') turf.distance(home, store, 'miles') turf.buffer(home, 3, 'miles') {% endhighlight %}

modules

{% highlight sh %} npm install turf-distance npm install turf-buffer {% endhighlight %} ..then call functions like this {% highlight js %} var distance = require('turf-distance') var buffer = require('turf-buffer') distance(home, store, 'miles') buffer(home, 3, 'miles') {% endhighlight %}
